# 🛡️ SOC Home Lab

## Overview

This repository documents my Security Operations Center (SOC) home lab, built to develop practical blue team skills in log analysis, threat detection, incident response, and SIEM monitoring.

The lab simulates enterprise environments using Windows and Linux systems while collecting and analyzing security events.

---

## Objectives

- Monitor security events
- Perform alert investigations
- Practice incident response
- Conduct threat hunting
- Improve SIEM skills
